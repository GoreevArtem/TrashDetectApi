# TrashDetectApi
Api для детектирования мусора на изображении

## Начало работы

Linux / Windows

предварительно нужно установить докер и make на рабочуюю машину
```
git clone https://github.com/GoreevArtem/TrashDetectApi
cd TrashDetectApi/
```

[//]: # (```)

[//]: # (pip install -r requirements.txt)

[//]: # (```)

### Запуск бэкенда

```
make up
```

### Убить бэкенд

```
make down
```
Остановить его

```
make stop
```
